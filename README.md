Análise de Dados de Arquivo CSV

Sobre o projeto
Este projeto tem como objetivo realizar a análise exploratória de dados a partir de um arquivo CSV, utilizando a linguagem Python e bibliotecas voltadas para ciência de dados.
A análise busca identificar padrões, tendências e possíveis insights relevantes contidos nos dados.
